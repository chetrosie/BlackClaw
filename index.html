About

关于这个项目

BlackClaw 是一个面向 OpenClaw / Clawdbot 托管场景的 MVP 项目，目标是验证 “营销站 + 计费事件 + 实例编排 + 运行状态可视化” 的完整链路。

项目目标

把复杂的 AI 运行环境标准化成可交付服务，让用户不再自己做部署与运维。

当前阶段

当前是可运行 MVP：支持模拟购买后自动创建实例并异步切换到运行状态。

后续计划

将 mock provider 与 mock billing 替换为真实云厂商 API 与 Stripe Webhook。

Why BlackClaw

为什么不是自建，而是托管

01

你省掉的是“隐形成本”

自建不仅是服务器费用，还包括排障时间、升级风险、运维负担和可用性波动。

02

你买到的是“持续可用”

BlackClaw 保证实例在线与可控，不需要你持续盯服务状态和日志告警。

03

你得到的是“可复制”

相同模板可在不同团队快速复制，避免每次从零搭环境导致标准不一致。

04

你专注的是“任务结果”

把注意力放在 AI 带来的产出上，而不是部署脚本、端口冲突和依赖兼容。
